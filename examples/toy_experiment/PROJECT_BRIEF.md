# Toy Experiment: MNIST Classification

## Goal
Train a simple CNN on MNIST to reach 99%+ accuracy.
This is a demonstration project for Deep Researcher Agent.

## Codebase
- Training script: train.py (to be created by the agent)
- Data: MNIST (auto-downloaded by torchvision)

## Constraints
- Single GPU (GPU 0)
- Max 10 epochs per run
- Use PyTorch
- Try at least 3 different learning rates

## Success Criteria
- Test accuracy > 99%
- Training completes without errors
